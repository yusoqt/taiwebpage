# My Projects Hub

ยินดีต้อนรับสู่พื้นที่รวมโปรเจกต์ของฉัน! 🌟 ไม่ว่าจะเป็นเว็บไซต์ MBTI Test ในธีม "หนังสือลึกลับ", โปรแกรมเข้ารหัส/ถอดรหัสไฟล์แบบง่าย, เว็บไซต์รวมคำถามลึกซึ้ง (สำหรับแฟน, เพื่อน, ครอบครัว หรือแม้แต่ถามตัวเอง!), โปรเจกต์จากรายวิชา และอีกหลากหลายไอเดียที่อยากแบ่งปัน 👍

## Features
- แบ่งโปรเจกต์เป็น 2 หมวด: **Personal** และ **Academic**
- มี Demo link สำหรับทดลองใช้งาน
- มี Source code บน GitHub
- กดรูปภาพเพื่อดูขยายได้ (Lightbox)

## Personal Projects
โปรเจกต์ที่ทำเองในเวลาว่าง ส่วนใหญ่เกิดจากความอยากลองหรืออยากใช้งานจริง เช่น เว็บแชท, ดูดวงไพ่ทาโร่, บอทเทรด Forex, QR Generator และอื่น ๆ

## Academic Projects
โปรเจกต์จากรายวิชาในมหาวิทยาลัย เช่น เกม Java (OOP), เว็บจองร้านอาหาร (Full-stack), และระบบ Smart Home ด้วย ESP32 + AI
